Bootstrap Features Used

This page lists all Bootstrap components, utilities, and layout systems used in my CA2 portfolio website.

Here is the link to view the components yourself: Bootstrap Documentation

1. Layout & Structure

  • Containers (.container)
  • Grid System (.row, .col-md-*)
  • Responsive Breakpoints
  • Spacing Utilities (p-*, m-*, g-*)
  • Responsive display utilities (d-flex, justify-content-center, etc.)

2. Navigation

  • Bootstrap Navbar
  • Navbar Toggler for mobile menus
  • Smooth scrolling
  • Sticky Navigation (.sticky-top)
  • Shadow utilities (.shadow-sm)

3. Hero Section

  • Buttons (.btn, .btn-primary)
  • Bootstrap spacing utilities for layout
  • gradient background

4. Skills Section (Tabs)

  • Bootstrap Tabs Component (.nav-tabs, .tab-pane)
  • Iconography using Bootstrap Icons
  • Cards (.card)

5. Education Section (Accordion)

  • Bootstrap Accordion Component (.accordion)
  • Collapse behavior (data-bs-toggle="collapse")

6. Projects Section

  • Bootstrap Carousel (.carousel)
  • arrow styling for carousel controls
  • Cards for project previews

7. Forms

  • Bootstrap Form Controls (.form-control)
  • Form layout utilities (.mb-3)
  • Full-width submit button (.w-100)

8. Modal Components

  • Modal for contact details
  • Message confirmation modal (triggered by JavaScript)
  • Modal triggers using data-bs-toggle + data-bs-target

9. Icons

  • Search — bi-search
  • Dark Mode — bi-moon
  • Light Mode — bi-sun
  • JavaScript — bi-braces
  • Bash — bi-terminal
  • C# — bi-code-slash
  • UI/UX Design — bi-brush
  • Graphs — bi-bar-chart
  • Teamwork — bi-people
  • Communication — bi-chat-dots
  • Email — bi-envelope
  • LinkedIn — bi-linkedin
  • Location — bi-geo-alt

10. Dynamic JavaScript Features

  • Smooth scrolling to sections
  • Back-to-top button with scroll detection
  • Dark mode toggle system
  • Loading spinner overlay
  • Modal popup confirmation for contact form

11. Additional UI Enhancements

  • Hover animations on cards
  • Rounded corner styling (.rounded)
  • Shadow effects (.shadow-sm, shadows)
  • Auto-open accordion for Year 4
  • Google Maps embedded via iframe
